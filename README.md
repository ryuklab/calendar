#Python Calendar using Tkinter

This project is a simple calendar application built with Python and Tkinter. It displays the current month and year, shows all dates in a calendar layout, and highlights the current day.

Features-

Displays the current month and year

Highlights today's date

Navigate between months using buttons

Simple and beginner-friendly Tkinter project


How it Works-

The application uses the calendar module to generate month data and datetime to get the current date. A custom draw_calendar() function creates and updates the calendar whenever the user changes months. Navigation buttons allow moving to the previous or next month while automatically handling year changes.
